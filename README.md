# CodeAlpha Network Sniffer Project

## Project Overview

The **Network Sniffer Project** is a Python-based tool designed to capture and analyze network packets.
This project helps in understanding how data travels through a network and provides basic insights into packet structures such as source IP, destination IP, protocol type, and payload data.

The tool is useful for learning **network monitoring, packet analysis, and cybersecurity fundamentals**.

## Objective

* Capture network packets in real time
* Analyze packet information
* Understand how data flows in a network
* Learn the basics of network security and packet sniffing

## Features

* Capture live network traffic
* Display source and destination IP addresses
* Identify protocols (TCP, UDP, ICMP)
* Display packet details
* Simple command-line interface

## Technologies Used

* **Programming Language:** Python
* **Libraries:** Scapy / Socket
* **Concepts:** Packet Sniffing, Network Monitoring, Cybersecurity

## Installation

### 1. Clone the repository

```
git clone https://github.com/your-username/CodeAlpha_Basic-Network-Sniffer.git
```

### 2. Navigate to the project directory

```
cd CodeAlpha_Basic-Network-Sniffer
```

### 3. Install required libraries

```
pip install scapy
```

### 4. Run the program

```
python network_sniffer.py
```

## Example Output

The program captures packets and displays information such as:
